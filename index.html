What is this?

REBORN-lang is the organization behind the REBORN programming language.

On this website you can find everything about REBORN-lang and REBORN.

The REBORN Programming Language

Reborn (stylized in all caps) is a general-purpose, statically typed, low-level programming language.

The REBORN-lang organization is actively working on making REBORN the best successor to C thanks to low-level

memory control and modernized straightforward syntax that is both familiar for experienced developers and easy for beginners.
